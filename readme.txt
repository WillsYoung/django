这个项目主要是django框架的使用，主要内容包括网站的登录/注册/退出登录，以及cookie的设置
以及设置登录以后才能进行的一系列操的实现。
其中包括了前后端分离的思想，就是后端只是负责后端的事情，比如处理数据，
与前端的交互就是传回一个或多个数据接口，并且将数据包装成如下格式
	{
	    ‘code'：0/200/xxx，
	     'data': {各种json格式的数据},
	     'msg' : '显示请求是否成功的信息'
	
	}
前端通过数据接口获取数据在渲染到网页上。
没有前后分离的app页面也在这个项目里面有所实现，通过django自己获取数据库的数据，然后自己解析到网页上面
